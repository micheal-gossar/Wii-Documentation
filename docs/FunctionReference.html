void GX_Init(void *fifo, u32 size)

Description

Initializes the GX graphics engine with the specified FIFO buffer. This function must be called before any other GX functions can be used. The FIFO buffer is used to queue graphics commands that are sent to the graphics processor.

Parameters

Parameter Type Description
fifo void* Pointer to a 32-byte aligned buffer that will serve as the command FIFO. Must not be NULL.
size u32 Size of the FIFO buffer in bytes. Minimum recommended size is 256KB (262144 bytes).

Return Value

None (void function)

Example Usage

#define DEFAULT_FIFO_SIZE (256 * 1024)

void initialize_graphics() {
    // Allocate 32-byte aligned FIFO buffer
    void *gp_fifo = memalign(32, DEFAULT_FIFO_SIZE);
    memset(gp_fifo, 0, DEFAULT_FIFO_SIZE);
    
    // Initialize GX with the FIFO buffer
    GX_Init(gp_fifo, DEFAULT_FIFO_SIZE);
    
    // Continue with other GX setup...
}

Notes

  • The FIFO buffer must be 32-byte aligned for optimal performance
  • A larger FIFO buffer can improve performance by reducing GPU stalls
  • This function should only be called once during application initialization
  • Call this function after VIDEO_Init() but before any drawing operations
