Getting Started with Wii GX

Last updated: 07/10/2025 | Author: Documentation Team

Overview

This guide walks you through setting up your development environment for Nintendo Wii GX graphics programming.

Prerequisites

  • devkitPPC toolchain installed
  • libogc with GX library
  • Basic C programming knowledge

Environment Setup


# Install devkitPro and devkitPPC
pacman -Syu devkitpro/devkitPPC devkitpro/libogc
# Set environment variables
export DEVKITPRO=/opt/devkitpro
export DEVKITPPC=$DEVKITPRO/devkitPPC

Project Structure


project/
  include/
    gx_init.h
  source/
    main.c
  Makefile

Building and Running


make
./build/game.elf
